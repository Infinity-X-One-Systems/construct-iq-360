## Blueprint Template Structure

Every construction project blueprint should include:

### 1. Cover Sheet
- Project name, address, owner
- Design professional and license
- Sheet index
- General notes and abbreviations
- Zoning and code information

### 2. Civil / Site Plans (C-Series)
| Sheet | Content |
|-------|---------|
| C-0.0 | Cover / Location Map |
| C-1.0 | Existing Conditions / Demo Plan |
| C-2.0 | Site Layout / Grading Plan |
| C-3.0 | Utilities Plan |
| C-4.0 | Stormwater Pollution Prevention Plan (SWPPP) |
| C-5.0 | Landscape Plan |

### 3. Architectural Plans (A-Series)
| Sheet | Content |
|-------|---------|
| A-0.0 | General Notes, Code Summary |
| A-1.0 | Floor Plan(s) |
| A-2.0 | Roof Plan |
| A-3.0 | Exterior Elevations (N/S/E/W) |
| A-4.0 | Building Sections |
| A-5.0 | Wall Sections |
| A-6.0 | Details |
| A-7.0 | Door & Window Schedules |
| A-8.0 | Finish Schedules |
| A-9.0 | Reflected Ceiling Plans |

### 4. Structural Plans (S-Series)
| Sheet | Content |
|-------|---------|
| S-0.0 | General Structural Notes |
| S-1.0 | Foundation Plan |
| S-2.0 | Framing Plans (each level) |
| S-3.0 | Structural Sections |
| S-4.0 | Structural Details |

### 5. MEP Plans
| Sheet | Content |
|-------|---------|
| M-1.0 | HVAC Floor Plan |
| M-2.0 | HVAC Schedules & Details |
| P-1.0 | Plumbing Floor Plan |
| P-2.0 | Plumbing Isometrics |
| E-1.0 | Electrical Floor Plan |
| E-2.0 | Electrical Panel Schedules |
| E-3.0 | One-Line Diagram |
| FP-1.0 | Fire Protection Plan |

## Residential Blueprint Checklist

### Minimum Required for Permit
- [ ] Site plan with setbacks and impervious surface calculation
- [ ] Floor plan with room labels and dimensions
- [ ] Exterior elevations (all 4 sides)
- [ ] Foundation plan
- [ ] Roof plan with pitch and materials
- [ ] Wall sections
- [ ] Energy calculations (Manual J/S/D for HVAC)
- [ ] Structural calculations (stamped PE)
- [ ] Truss layout (if applicable)

### Florida Specific Requirements
- [ ] Florida Product Approval numbers for windows/doors
- [ ] High-velocity hurricane zone (HVHZ) compliance if Miami-Dade
- [ ] Wind speed calculations (150+ mph for most of FL)
- [ ] Flood zone compliance (FEMA FIRM map)
- [ ] Energy code compliance (Florida Energy Code)

## Blueprint Review Process

### Pre-Construction Review
1. **Completeness Check** ‚Äî All sheets present per index
2. **Coordination Check** ‚Äî MEP vs. structural conflicts
3. **Code Compliance** ‚Äî Building code, fire code, accessibility
4. **Constructability Review** ‚Äî Feasibility, conflicts, missing details
5. **Permit Submittal** ‚Äî Submit to AHJ (Authority Having Jurisdiction)

### Field Blueprint Management
- Issued For Construction (IFC) set on site at all times
- All addenda and revisions incorporated
- Approved submittals attached to relevant sections
- RFI responses logged
