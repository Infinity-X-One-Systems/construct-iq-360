## 1. Pre-Construction Checklist

### Permits & Approvals
- [ ] Building permit obtained and posted
- [ ] Electrical permit obtained
- [ ] Plumbing permit obtained
- [ ] Mechanical / HVAC permit obtained
- [ ] Fire sprinkler permit obtained
- [ ] Civil / grading permit obtained
- [ ] Environmental permits (wetlands, stormwater)
- [ ] HOA / ARB approval (residential)
- [ ] Utility locates completed (811 — Call Before You Dig)
- [ ] Right-of-way permits (if work in street)
- [ ] Notice of Commencement recorded

### Contracts & Insurance
- [ ] Owner-Contractor agreement signed
- [ ] All subcontractor agreements executed
- [ ] Owner insurance requirements met and certificates issued
- [ ] GC GL, Workers' Comp, Auto, Builder's Risk certificates current
- [ ] All subcontractor certificates collected and compliant
- [ ] Payment and Performance Bonds obtained (if required)
- [ ] Lien rights notice served (if required by state)

### Site Preparation
- [ ] Survey stakes in place
- [ ] Geotechnical report reviewed
- [ ] Demolition complete (if applicable)
- [ ] Existing conditions photographed
- [ ] Temporary power installed
- [ ] Temporary water installed
- [ ] Construction fence / barricade installed
- [ ] Signage posted (permit card, contractor info)
- [ ] Erosion and sediment control installed (SWPPP)
- [ ] Tree protection in place
- [ ] Staging area established
- [ ] Temporary restrooms on site

### Pre-Construction Meeting
- [ ] Owner / developer attended
- [ ] Architect / engineer attended
- [ ] All major subcontractors attended
- [ ] Schedule reviewed and accepted
- [ ] Communication protocol established
- [ ] Meeting minutes distributed within 48 hours

---

## 2. Foundation Inspection Checklist

- [ ] Excavation depth and dimensions verified
- [ ] Soil bearing capacity confirmed (geotechnical)
- [ ] Formwork properly braced and aligned
- [ ] Rebar size, spacing, and coverage as specified
- [ ] Anchor bolts / hold-downs placed per plans
- [ ] Penetrations sleeved
- [ ] Utility under-slab rough-ins placed and inspected
- [ ] Vapor barrier installed
- [ ] Ready-mix concrete ordered per specifications (mix design, PSI)
- [ ] Building department inspection passed
- [ ] Concrete poured and properly cured

---

## 3. Framing / Rough Inspection Checklist

- [ ] Foundation anchor bolts verified
- [ ] Sill plates pressure-treated and sealed
- [ ] Wall framing per plans (stud spacing, headers, king/jack studs)
- [ ] Structural steel / beams placed per plans
- [ ] Floor framing per plans (joist spacing, blocking, strapping)
- [ ] Roof framing per plans (rafter/truss spacing, ridge, hips/valleys)
- [ ] Hurricane straps / hold-downs installed
- [ ] Shear panels nailed per schedule
- [ ] Bracing installed
- [ ] Rough opening sizes match door/window schedule
- [ ] Fire blocking installed at required locations
- [ ] Building department framing inspection passed

---

## 4. MEP Rough-In Checklist

### Rough Plumbing
- [ ] DWV (drain, waste, vent) stack sizes per plans
- [ ] Clean-outs installed at required locations
- [ ] Pressure test performed and passed
- [ ] Water supply lines sized and routed per plans
- [ ] Stub-outs at all fixture locations
- [ ] Plumbing rough inspection passed

### Rough Electrical
- [ ] Panel location approved
- [ ] Service entrance sized per load calculation
- [ ] Branch circuits routed per plans
- [ ] Box locations match switch/outlet schedule
- [ ] AFCI / GFCI circuits identified
- [ ] Electrical rough inspection passed

### Rough HVAC
- [ ] Duct layout per Manual D
- [ ] Return air and supply registers per equipment schedule
- [ ] Equipment pads / curbs installed
- [ ] Refrigerant lines roughed in
- [ ] HVAC rough inspection passed

### Rough Fire Protection
- [ ] Sprinkler heads at required locations
- [ ] Pipe sizes per hydraulic calculations
- [ ] Pressure test performed and passed
- [ ] Fire protection rough inspection passed

---

## 5. Insulation Checklist

- [ ] Walls: R-value per energy code (min R-15 FL)
- [ ] Ceilings: R-value per energy code (min R-38 FL)
- [ ] All penetrations sealed (air barrier)
- [ ] Vapor retarder installed (if required)
- [ ] Insulation inspection passed

---

## 6. Drywall / Interior Checklist

- [ ] Fire-rated assemblies per plans (1-hour, 2-hour walls)
- [ ] Moisture-resistant board in wet areas
- [ ] Corner bead installed
- [ ] All screw/nail spacing per fire-rated assembly
- [ ] Tape, mud, and finish per paint spec (Level 4 or 5)
- [ ] Drywall inspection passed (if required)

---

## 7. Finishes Checklist

- [ ] Flooring installed (no cuts, consistent grout joints, transitions)
- [ ] Cabinets level and plumb, doors and drawers operating
- [ ] Countertops installed and sealed
- [ ] Tile work — grout sealed, no lippage, consistent joints
- [ ] Paint — final coat, no holidays, touch-ups complete
- [ ] Trim — caulked, painted, no gaps
- [ ] Doors and hardware — operating, aligned, latching
- [ ] Windows — operating, sealed, screens installed
- [ ] Plumbing fixtures — installed, leaks tested
- [ ] Electrical devices — installed, plates on, tested
- [ ] HVAC trim-out — grilles, thermostats, balanced

---

## 8. Project Closeout Checklist

### Required Documents
- [ ] Certificate of Occupancy (CO) obtained
- [ ] All inspection sign-offs collected
- [ ] As-built drawings completed
- [ ] Equipment manuals / warranties delivered
- [ ] Attic/access key delivered
- [ ] Subcontractor lien releases collected (final, unconditional)
- [ ] Final waiver signed by contractor
- [ ] Punchlist completed and signed off

### Final Walkthrough
- [ ] Owner walkthrough completed
- [ ] All punchlist items resolved
- [ ] Cleaning complete
- [ ] All debris removed from site
- [ ] Final photos taken

### Training
- [ ] HVAC system operation explained
- [ ] Sprinkler system operation explained
- [ ] Electrical panel labeled
- [ ] Emergency shutoffs identified

---

## 9. Quality Control Inspection (Daily)

- [ ] Work matches approved drawings and submittals
- [ ] Materials match approved samples
- [ ] Workmanship meets specification standards
- [ ] RFIs answered before proceeding on unclear items
- [ ] No unauthorized substitutions
- [ ] Safety compliance observed
- [ ] Daily log updated
